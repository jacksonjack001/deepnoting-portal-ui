Codex 安装配置

把 Codex 配置为通过AI Proxy网关调用 gpt-5.5。请把 YOUR_KEY 替换为门户里分配给你的 Key。

安装 Codex

平台 安装方式
macOS / Linux curl -fsSL https://chatgpt.com/codex/install.sh | sh
Windows PowerShell powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"
npm 通用安装 npm install -g @openai/codex
Homebrew brew install --cask codex

配置环境变量

下面配置中的 env_key = "AI_PROXY_API_KEY" 会读取同名环境变量。真实 Key 只放在本机用户环境变量里，不要提交到项目仓库。

# macOS / Linux / WSL
export AI_PROXY_API_KEY="YOUR_KEY"
# Windows PowerShell，设置后重新打开终端
[Environment]::SetEnvironmentVariable("AI_PROXY_API_KEY", "YOUR_KEY", "User")
:: Windows CMD，设置后重新打开终端
setx AI_PROXY_API_KEY "YOUR_KEY"

创建配置文件

配置文件位置：macOS / Linux 为 ~/.codex/config.toml，Windows 为 %USERPROFILE%\.codex\config.toml。

# macOS / Linux / WSL
mkdir -p ~/.codex
nano ~/.codex/config.toml
# Windows PowerShell
New-Item -ItemType Directory -Force "$env:USERPROFILE\.codex"
notepad "$env:USERPROFILE\.codex\config.toml"

config.toml

model_provider = "ai-proxy"
model = "gpt-5.5"
model_reasoning_effort = "medium"
plan_mode_reasoning_effort = "xhigh"
model_reasoning_summary = "none"
model_context_window = 1050000
model_auto_compact_token_limit = 945000
tool_output_token_limit = 6000
approval_policy = "never"
sandbox_mode = "danger-full-access"
suppress_unstable_features_warning = true
cache_ttl = "30m"

[model_providers.ai-proxy]
name = "OpenAI"
base_url = "https://llm.example.com"
wire_api = "responses"
requires_openai_auth = false
env_key = "AI_PROXY_API_KEY"
supports_websockets = false

[features]
responses_websockets_v2 = false
multi_agent = true
remote_connections = true

[agents]
max_threads = 4
max_depth = 1
job_max_runtime_seconds = 1800

[tui]
status_line = ["model-with-reasoning", "current-dir"]

[tui.model_availability_nux]
"gpt-5.5" = 4

验证

codex --version
codex

注意事项

Key 不入库不要把真实 Key 写入项目文件、截图或公开文档。
危险权限approval_policy = "never" 和 sandbox_mode = "danger-full-access" 只建议在可信机器和可信项目中使用。
缓存生效修改环境变量后，重新打开终端再启动 Codex。
