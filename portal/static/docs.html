API 调用文档

支付完成后，使用注册时得到的 Key 调用 OpenAI 兼容接口。

Base URL

https://llm.example.com

基础 Chat Completions

curl https://llm.example.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "qwen3-32b",
    "messages": [
      {"role": "system", "content": "你是一个简洁可靠的助手。"},
      {"role": "user", "content": "你好，帮我写一个注册欢迎语。"}
    ]
  }'

Python SDK

from openai import OpenAI

client = OpenAI(
    api_key="YOUR_KEY",
    base_url="https://llm.example.com"
)

response = client.chat.completions.create(
    model="qwen3-32b",
    messages=[{"role": "user", "content": "你好"}],
    timeout=45,
)
print(response.choices[0].message.content)

JavaScript SDK

import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.AI_PROXY_API_KEY,
  baseURL: "https://llm.example.com",
});

const response = await client.chat.completions.create({
  model: "qwen3-32b",
  messages: [{ role: "user", content: "生成一个 5 条 TODO 列表" }],
});

console.log(response.choices[0].message.content);

流式输出

from openai import OpenAI

client = OpenAI(api_key="YOUR_KEY", base_url="https://llm.example.com")

stream = client.chat.completions.create(
    model="qwen3-32b",
    messages=[{"role": "user", "content": "分步骤解释 HTTP 缓存"}],
    stream=True,
)

for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="")

最佳实践

Key 放服务端不要把 Key 放到网页、App 包或公开仓库中。
记录 request_id排查异常时保留请求时间、模型、request_id 和返回状态。
限制上下文长对话要摘要旧消息，避免无意义 token 消耗。
设置超时生产调用建议设置 30-60 秒超时，并对 429/5xx 做有限重试。
模型分层轻量任务优先低成本模型，复杂推理再使用高阶模型。
监控余额接近预算上限时及时续费，避免线上服务被阻断。
