Claude Code 安装配置

把 Claude Code 配置为通过AI Proxy网关调用 claude-opus-4-8。请把 YOUR_KEY 替换为门户里分配给你的 Key。

安装 Claude Code

平台 安装方式
macOS / Linux / WSL curl -fsSL https://claude.ai/install.sh | bash
Windows PowerShell irm https://claude.ai/install.ps1 | iex
Windows CMD curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
Homebrew brew install --cask claude-code
WinGet winget install Anthropic.ClaudeCode
npm 通用安装 npm install -g @anthropic-ai/claude-code

配置环境变量

ANTHROPIC_AUTH_TOKEN 会作为 Bearer Token 发给网关。真实 Key 只放在本机用户环境变量或用户级 settings.json 中。

# macOS / Linux / WSL
export AI_PROXY_API_KEY="YOUR_KEY"
export ANTHROPIC_BASE_URL="https://example.com/portal/anthropic"
export ANTHROPIC_AUTH_TOKEN="$AI_PROXY_API_KEY"
# Windows PowerShell，设置后重新打开终端
[Environment]::SetEnvironmentVariable("AI_PROXY_API_KEY", "YOUR_KEY", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://example.com/portal/anthropic", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", "YOUR_KEY", "User")
:: Windows CMD，设置后重新打开终端
setx AI_PROXY_API_KEY "YOUR_KEY"
setx ANTHROPIC_BASE_URL "https://example.com/portal/anthropic"
setx ANTHROPIC_AUTH_TOKEN "YOUR_KEY"

创建配置文件

用户级配置文件位置：macOS / Linux 为 ~/.claude/settings.json，Windows 为 %USERPROFILE%\.claude\settings.json。如果写项目级配置，避免把真实 Key 放进会提交的 .claude/settings.json。

# macOS / Linux / WSL
mkdir -p ~/.claude
nano ~/.claude/settings.json
# Windows PowerShell
New-Item -ItemType Directory -Force "$env:USERPROFILE\.claude"
notepad "$env:USERPROFILE\.claude\settings.json"

settings.json

{
  "env": {
    "ANTHROPIC_BASE_URL": "https://example.com/portal/anthropic",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_KEY",
    "API_TIMEOUT_MS": "3000000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1",
    "ANTHROPIC_MODEL": "claude-opus-4-8",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "claude-opus-4-8",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "claude-opus-4-8",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "claude-opus-4-8",
    "CLAUDE_CODE_MAX_OUTPUT_TOKENS": "128000"
  },
  "skipWebFetchPreflight": true,
  "skipDangerousModePermissionPrompt": true
}

验证

claude --version
claude

注意事项

Key 不入库不要把真实 Key 写入项目文件、截图或公开文档。
权限提示skipDangerousModePermissionPrompt 会跳过危险模式提示，只建议在可信机器和可信项目中使用。
TPM 匹配CLAUDE_CODE_MAX_OUTPUT_TOKENS=128000 需要 160K 或更高 TPM 档位；30K TPM 请把该值降到 30000 以下。
环境生效修改系统环境变量后，重新打开终端再启动 Claude Code。
